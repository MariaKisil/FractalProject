# Аналіз МРТ-знімків

## Опис проєкту
Програмний комплекс призначений для автоматизованого фрактального аналізу цифрових медичних зображень (МРТ). Програма дозволяє диференціювати здорові тканини та злоякісні новоутворення на основі їхньої топологічної складності за допомогою методів:
- Метод підрахунку квадратів
- Метод покриття (розмірність Гаусдорфа)
- Метод «маса-радіус»
- Метод дилатації пікселів

## Вимоги до системи
Для коректної роботи програми необхідний інтерпретатор **Python 3.8** або новішої версії, а також наступні бібліотеки:
- `numpy`
- `opencv-python`
- `Pillow`
- `matplotlib`
- `scipy`
- `tk`

## Встановлення та запуск
1. Клонуйте або завантажте папку з проєктом.
2. Рекомендується створити віртуальне середовище:
   python -m venv venv
   source venv/bin/activate  # Для Linux/Mac
   venv\Scripts\activate     # Для Windows
3. Встановіть необхідні залежності:
   pip install -r requirements.txt
4. Запустіть головний файл програми:
   python main.py

## Структура проєкту
- `main.py` — точка входу, запуск графічного інтерфейсу.
- `fractal_manager.py` — обчислювальне ядро з реалізацією математичних методів.
- `preprocessing.py` — алгоритми попередньої обробки..
- `/data` — папка для тестових МРТ-знімків.
